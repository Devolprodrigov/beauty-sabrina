# Sabrina Beauty - versão pronta para GitHub Pages

Esta versão foi convertida para **HTML + CSS + JavaScript puro**, então você pode publicar direto no GitHub sem Vite, npm ou build.

## Como publicar

1. Crie um repositório no GitHub.
2. Envie estes arquivos para a raiz do repositório:
   - `index.html`
   - `styles.css`
   - `script.js`
3. No GitHub, vá em **Settings > Pages**.
4. Em **Build and deployment**, escolha:
   - **Source:** Deploy from a branch
   - **Branch:** `main` / `/root`
5. Salve.
6. Aguarde o GitHub gerar o link do site.

## O que já está funcionando

- catálogo de produtos
- carrinho
- pedido por WhatsApp
- painel de gestão
- cadastro e edição de produtos
- estoque salvo no navegador com `localStorage`
- exportação simples em PDF

## Observações

- O estoque fica salvo no navegador do aparelho usado.
- Se quiser depois, eu também posso te entregar uma **versão com login/admin protegido** e integração com banco de dados.
